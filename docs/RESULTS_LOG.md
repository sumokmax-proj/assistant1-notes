## 기록 형식

```markdown
### [YYYY-MM-DD] 실험명
- **가설:** 무엇이 될 거라고 생각했는가
- **행동:** 실제로 무엇을 했는가
- **결과:** 수치 포함한 실제 결과
- **학습:** 이 결과에서 배운 것
- **다음:** 이 결과 기반으로 무엇을 바꿀 것인가
```
