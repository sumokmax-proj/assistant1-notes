## 1순위 후보: 디지털 제품 판매 (Gumroad)

### 개요
AI가 실용적인 디지털 제품(코드 템플릿, 자동화 스크립트, 체크리스트 등)을 만들어 Gumroad에서 판매.

### 왜 이게 최우선인가
- **초기 비용 $0:** Gumroad 무료 플랜 존재 (수수료 10% 과금)
- **즉시 시작 가능:** 계정 생성 후 당일 업로드 가능
- **AI 강점 직접 활용:** 코드 작성, 문서화, 번들링 모두 AI가 가능
- **검증된 시장:** Gumroad에 이미 구매자 존재

### 판매 가능한 제품 예시

```
카테고리           제품 예시                       가격대
──────────────────────────────────────────────────────
코드 템플릿       Python 자동화 스크립트 팩        $5~15
                  Node.js API 보일러플레이트         $10~25
Notion 템플릿     프리랜서 업무 관리 템플릿          $5~10
프롬프트 팩       ChatGPT 마케팅 프롬프트 100개      $5~15
체크리스트        스타트업 런치 체크리스트            $3~8
데이터 도구       Excel/Sheets 자동화 매크로          $8~20
```

### 실행 단계
1. Operator: Gumroad 계정 생성 (무료)
2. Claude: 제품 선정 + 콘텐츠 제작
3. Operator: Gumroad에 업로드 및 결제 수단 연동 (승인 필요)
4. Claude: 마케팅 문구 작성, SEO 최적화
5. 결과 측정 → 개선 반복

- 계약서 요약 API (소상공인 타겟)

### 필요 조건
